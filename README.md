# My Neovim Config

Lua based neovim config. 

This configuration requires Neovim 0.7+, due to [Legendary](https://github.com/mrjones2014/legendary.nvim).

Bootstrappping packer is not working, please install manually following the instructions in the main repository.

## Install 

```
git clone https://github.com/guilherme-puida/nvim ~/.config/nvim
```
